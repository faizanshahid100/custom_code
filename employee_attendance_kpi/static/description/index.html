📊 Employee Attendance & KPI Tracking

🎯 Overview

A comprehensive Odoo module that combines employee attendance tracking with Key Performance Indicator (KPI) monitoring. Automatically creates daily records for all employees and calculates weighted KPI performance based on assigned targets.

✨ Key Features

📅 Smart Attendance Tracking

Automatically determines attendance status:

  • Present Active attendance
  • Absent No record found
  • Leave Approved leave
  • Weekend Non-working day
  • Gazetted Public holiday

📈 KPI Monitoring

Track four key performance metrics:

  • 🎫 Resolved Tickets
  • 📞 Calls
  • ⏰ Billable Hours
  • ⚡ Response Time

⚖️ Weighted Performance

Intelligent KPI percentage calculation based on assigned metrics

KPIs Weight
1 KPI100%
2 KPIs50% each
3 KPIs33.33% each
4 KPIs25% each

🤖 Automation

Daily automated record creation:

  • Runs every day at 1:00 AM
  • Creates records for all active employees
  • Fetches KPI data from daily.progress
  • Calculates performance metrics

🎨 Multiple Views

  • List View: Color-coded attendance overview with KPI summaries
  • Form View: Detailed daily record with KPI breakdown and targets
  • Pivot View: Analyze performance across employees and time periods
  • Graph View: Visualize KPI trends over time
  • Calendar View: Monthly attendance calendar

⚙️ Configuration

Set Employee KPI Targets

Navigate to: HR → Employees → KPI Targets Tab

Configure daily targets for each employee. Set to 0 to disable a KPI.

Example Configurations:

Role Tickets Calls Billable Hours Response Time
Support Agent 10 30 0 1.5h
Developer 3 0 7 0
Sales Rep 0 50 0 0

🔌 Integration

Integrates seamlessly with:

  • daily.progress model for KPI data
  • hr.attendance for presence tracking
  • hr.leave for leave management
  • resource.calendar.leaves for holidays

📦 Installation

  1. Copy module to Odoo addons directory
  2. Update Apps List
  3. Search for "Employee Attendance & KPI Tracking"
  4. Click Install
  5. Configure employee KPI targets
  6. Done! Records will be created automatically

📋 Requirements

  • Odoo 14.0+ (compatible with 14, 15, 16, 17)
  • HR module installed
  • daily.progress model (or similar) for KPI data source
